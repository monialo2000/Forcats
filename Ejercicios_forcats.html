Factores que son

Los factores se usan para trabajar con variables categòricas que tienen un numero determinado de valores posibles. Generalmente son códigos, cadenas. Esos códigos si se mantienen como caracteres se ordenan de manera alfabética, que posiblemente no sea el orden deseado. Un factor es un vector de enteros con un atributo de niveles que guarda mapeos entre enterosy valores categóricos. Anteriormente cuando uno leìa en base un archivo solìa dejar las variables tipo char como categoricas por default parámetro “stringsAsFactors” en read. En tidyverse eso no sucede.

x1 <- c("Dic", "Abr", "Ene", "Mar")
niveles_meses <- c("Ene", "Feb", "Mar", "Abr", "May", "Jun",
  "Jul", "Ago", "Sep", "Oct", "Nov", "Dic")
y1 <- factor(x1, levels = niveles_meses)
y1
## [1] Dic Abr Ene Mar
## Levels: Ene Feb Mar Abr May Jun Jul Ago Sep Oct Nov Dic
sort(x1)
## [1] "Abr" "Dic" "Ene" "Mar"
sort(y1)
## [1] Ene Mar Abr Dic
## Levels: Ene Feb Mar Abr May Jun Jul Ago Sep Oct Nov Dic
f <- factor(c("a", "c", "b", "a"),
levels = c("a", "b", "c"))
f

## Levels: a b c

convert_income <- function(x) {
  regex <- "^(?:Lt |)[$]([0-9]+).*$"
  is_range <- grepl(regex, x)
  num_income <- as.numeric(gsub(regex, "\\1", x[is_range]))
  num_income <- trunc(num_income / 5000) * 5000
  x[is_range] <- paste0("Gt $", num_income)
  x
}
